Step-by-step guide

1

Install UserLAnd

Download and install UserLAnd from the Google Play Store. This app allows you to run Linux distributions on your Android device.

Download from Google Play
2

Set up Ubuntu

Open UserLand, choose Ubuntu → Minimal → Terminal. This will set up a lightweight Ubuntu environment on your device.

3

Wait for installation

Wait for distribution to install. This may take a few minutes depending on your internet connection. You'll also see windows with [y/n]just type y in meantime, you can pour some tea ;)

4

Paste installation command

After successful installation, a terminal will open in front of you, write there:

sudo apt update && sudo apt upgrade -y && sudo apt install python3 git python3-pip -y && git clone https://github.com/coddrago/Heroku && cd Heroku && sudo pip install -r requirements.txt && python3 -m heroku --no-web
5

Login to your account

You will be asked to enter your account information, follow the instructions and read labels carefully. If you don't know English well, watch the video guide or use a translator.

1

Get Telegram API Credentials

  • Go to https://my.telegram.org and log in with your Telegram account.
  • Click on "API development tools".
  • Create a new application by filling in required details (any name, short description and url will do).
  • Copy your API ID and API hash.
2

Provide Credentials to terminal in userland

  • When prompted, enter your copied API ID (e.g., 33728218).
  • Next, enter your copied API hash (e.g., 2c416c3a6e867f9c67169a6c8506990b).
3

Choose Login Method

  • Decide if you want to use a QR code for login - type y for yes, or n for phone number login.
  • If you chose phone, enter your number in international format (e.g., +7908 ***).
  • Once complete, your API config will be saved and the userbot will log in.
6

All done!

Voila! You have installed Heroku on UserLAnd. Everything is ready to use.
